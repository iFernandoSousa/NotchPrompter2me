# App icon

Add `icon.png` here (e.g. 256×256 or 512×512) to use a custom app icon.

- **Development (`npm start`)**: If `icon.png` exists, it will be used for the dock icon on macOS.
- **Packaging**: For a built app, add `icon.icns` (macOS) and/or `icon.ico` (Windows) so Electron Forge can embed them.
